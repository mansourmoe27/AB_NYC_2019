# AB_NYC_2019

# New York City Airbnb 2019 Analysis

## Project Overview

This project analyzes Airbnb listings in New York City using Python and data visualization techniques. The goal was to better understand how factors such as neighborhood, room type, availability, and reviews affect Airbnb pricing.

As someone transitioning into technology and data analysis, I wanted to work with a real-world dataset instead of a small classroom example. This project helped me strengthen my skills in data cleaning, exploratory data analysis, and visualization.

---

## Dataset

The project uses the AB_NYC_2019 dataset, which contains thousands of Airbnb listings across New York City's five boroughs.

Features include:

- Listing price
- Neighborhood group
- Room type
- Number of reviews
- Availability
- Latitude and longitude
- Host information

---

## Technologies Used

---

## Challenges I Faced

One of the biggest challenges was working with a large dataset and deciding which variables were actually useful for analysis. At first, I focused on too many columns and found it difficult to identify meaningful trends.

Another challenge was handling outliers. Some Airbnb listings had extremely high prices that distorted the visualizations and made it harder to understand the typical market. I had to learn how to investigate unusual values and determine whether they should be included in the analysis.



---

## Key Questions Explored

- Which neighborhoods have the highest Airbnb prices?
- How do room types impact pricing?
- Which areas have the highest concentration of listings?
- Are highly reviewed listings generally more expensive?
- How does availability vary across neighborhoods?

---

## Key Findings

- Manhattan listings generally had higher average prices than other boroughs.
- Entire homes and apartments were typically more expensive than private rooms.
- Listing density varied significantly between neighborhoods.
- Price distributions contained several extreme outliers.
- Popular tourist areas tended to have higher-priced listings.
